Estrutura Básica

A estrutura básica de um documento HTML inclui a declaração do tipo de documento !DOCTYPE html, seguida pela tag html que envolve todo o conteúdo. Dentro dela, há duas seções principais: head e body. O head contém metadados como título do documento title, links para folhas de estilo e scripts, enquanto o body inclui o conteúdo visível na página, como texto, imagens e outros elementos.

Tags

Tags em HTML são comandos que definem a estrutura e o conteúdo de uma página web. Elas são delimitadas por colchetes angulares, por exemplo, tagname>. As tags vêm em pares de abertura e fechamento, como p>p para parágrafos, ou em formato auto-fechado, como img para imagens. Cada tag possui uma função específica e pode conter outros elementos ou atributos.

Atributos de Tags

Atributos são usados para fornecer informações adicionais sobre elementos HTML. Eles são especificados dentro da tag de abertura e têm a forma nome="valor". Por exemplo, na tag a href="https://example.com", href é um atributo que define o URL do link. Outros exemplos incluem class, id, style e src.

Textos - Tipografias

Tipografia em HTML refere-se ao estilo e apresentação do texto na página. As tags básicas para formatação de texto incluem h1> a h6> para cabeçalhos, p> para parágrafos, b> e strong> para negrito, i> e em> para itálico, e u> para sublinhado. Além disso, o CSS (Cascading Style Sheets) é frequentemente usado para controlar a aparência da tipografia, como fontes, tamanhos e cores.

Listas

Listas são usadas para agrupar itens relacionados e podem ser ordenadas ou não ordenadas. Uma lista não ordenada é criada com a tag ul>, contendo itens de lista li>, que são geralmente marcados com pontos. Já uma lista ordenada usa a tag ol>, e os itens de lista li> são numerados automaticamente. As listas podem ser aninhadas para criar sublistas.

Links

Links, ou âncoras, são elementos interativos que permitem a navegação entre páginas ou recursos. A tag a> é usada para definir um link, com o atributo href especificando o destino. Por exemplo, a href="https://example.com">Clique aqui/a>. Links podem ser estilizados com CSS e podem abrir em novas abas ou janelas usando o atributo target="_blank".
